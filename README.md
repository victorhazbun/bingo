## Instructions

### Front-end

* Installing dependencies `npm install`

* Compile the Elm file and open the web app `elm-live Bingo.elm --open --output=bingo.js`

### Back-end

* Navigate to the server directory `cd server`

* Installing dependencies `npm install`

* Start the server `node server.js`
